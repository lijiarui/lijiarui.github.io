这是一个设计预览。所有数字都是 mock 的，不代表真实统计。

下一步把它连真实数据，三条路径供选：

  • Umami Cloud（推荐自定义大屏）：免费 10 万事件/月，有 REST API，可以把数据直接拉进这个页面。一行 beacon 脚本到博客 head
  • Cloudflare Web Analytics：免费、隐私友好、零配置；juzi.bot 应该已经在 CF 上了，开关一开即用。但只能用 CF 自己的面板，没公开 API 给自定义大屏
  • 百度统计 + 谷歌 GA4 双埋：覆盖国内 + 国外，最经典；但要做自定义大屏需要走 OAuth + Reporting API，最复杂

如果选 Umami Cloud，我可以下一步帮你：① 在博客所有页面 head 加一行 beacon ② 把这个大屏的 mock 数据替换成 Umami API 调用 ③ 部署到比如 https://rui.juzi.bot/dashboard/ 加密码保护或 noindex。
